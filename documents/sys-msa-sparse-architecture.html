MSA: 迈向一亿 Token 的终极记忆架构

长短期记忆是人类智能的基石。然而，当今大语言模型的上下文极限仍受困于算力与架构的桎梏。抛弃传统的外部检索补丁，这是一种秩序优先、绝对可扩展的内在记忆系统。

01

认知缺口与困境

认知科学研究表明，人类记忆的功能性信息容量预估在 109 比特左右。假设每个 Token 的有效语义密度为 3 至 5 比特，这相当于约 2 亿至 3 亿个 Token 的终身容量。

要真正弥合人工智能与人类规模记忆之间的鸿沟，并催生如稳定人格的数字孪生（Digital Twins）或长历史多智能体推理等复杂应用，模型必须具备处理上亿 Token 级别上下文的能力。然而，当前的现实是骨感的：即使是明确为长上下文设计的顶尖大语言模型（LLMs），其有效长度极限通常也停留在 100 万 Token 的阈值。

现有的探索在应对极限长度时，往往暴露出致命的缺陷。混合线性注意力、固定大小记忆状态（如 RNNs）或是外部存储系统（如 RAG），在上下文规模暴增时，不仅面临着严重的精度衰减、延迟激增，还缺乏端到端的优化能力，甚至无法动态修改记忆内容。这成为了限制 AI 进化为“数字生命”的算力与架构枷锁。

三种流派，一种答案。

当前的 LLM 长期记忆研究主要分为三大范式，但只有重构注意力机制，才能实现绝对的可扩展性与高保真度。

Paradigm I

参数化记忆
Parameter-Based

直接更新模型参数（如 LoRA 或 CPT）或利用测试时训练（如 Titans）。虽然具备高语义整合度，但根本上缺乏容量扩展性，极易遭遇**灾难性遗忘（Catastrophic Forgetting）**，且训练开销高昂。

Paradigm II

外部存储记忆
External Storage

以 RAG 和 MemAgent 为代表。容量可自然扩展至终身级别且无遗忘问题，但由于依赖离散的语义表示（如文本块或外部向量），无法实现端到端的可微性优化，导致检索精度存在固有天花板。

Paradigm III

潜在状态记忆
Latent State

在模型原生表示空间内构建记忆（如 DeltaNet、RWKV）。线性注意力通过将历史压缩至固定大小状态来实现极高效率，但在极端长度下这种有损压缩必定导致严重的性能衰减与信息丢失。

The Solution

MSA (Ours)
Memory Sparse Attention

端到端可训练、高检索精度、完美的 𝒪(L) 线性复杂度。通过解耦记忆容量与推理能力，既保留了潜在状态记忆的高保真度，又赋予了模型扩展至一亿 Token 的终身记忆生命力。

02

记忆稀疏注意力

彻底打破稠密注意力的算力桎梏。

为在潜在状态级别高效处理海量记忆，MSA 将标准的稠密自注意力替换为基于文档检索的**稀疏注意力机制（Sparse Attention Mechanism）**。其核心理念是：将记忆空间的检索与最终答案的生成无缝融合为一个端到端优化的统一架构，超越传统“先检索后阅读”的流水线局限。

路由与压缩机制

除了利用骨干模型的投影权重生成标准的键（K）与值（V）矩阵外，MSA 并行引入了一个参数化的 Router K Projector 来生成专门的路由键矩阵 KR。为将内存占用与检索复杂度降至最低，文档被分割为定长块，并执行均值池化（Mean Pooling）将其压缩为极简的潜在表示。

生成式组装

推理阶段，通过计算 Query 路由向量与记忆库压缩路由键的余弦相似度，模型筛选出 Top-k 的高相关度文档。随后，仅加载被选中片段的压缩 KV 矩阵，拼接至当前 Query 的局部缓存之前，最后执行稀疏感知的自回归生成。

解耦位置语义：并行与全局 RoPE

扩展记忆域面临的一个致命挑战是**训练集（短）与推理集（极长）之间的分布差异**。标准全局位置编码会导致文档数量增多时位置 ID 剧烈偏移，引发性能崩塌。MSA 为每一个入库文档分配独立的 Position ID（均从 0 起始），彻底将位置语义与总文档数解耦；与此同时，对当前活跃的 Query 序列采用考虑检索偏移量的全局 RoPE，从而完美实现了在 64K 上下文上训练，却能零散失泛化至 1 亿 Token 推理的奇迹。

极限工程与推理

如何在单节点 2 × A800 上吞噬一亿 Token？

理论架构必须服从物理硬件的约束。即使采用池化压缩，1 亿 Token 的 KV 缓存仍需约 169GB 内存，这已突破标准双卡 A800 的显存极限。为此，MSA 设计了**分层存储策略与记忆交织机制**。

Stage 1: 离线全局编码 (Offline)

一次性将海量文档库前向传播，生成并压缩出紧凑的 K、V 与路由键矩阵。这一 𝒪(LG) 复杂度的重度运算被完美摊销为离线操作。

Stage 2: 分层存储与在线路由 (Memory Parallel)

为了打破显存瓶颈，**路由键驻留于 GPU 显存**以保障毫秒级的相似度分布打分；而**海量内容 KVs 则被卸载至 CPU DRAM**。仅当确切的 Top-k 块被选中时，才异步拉取至 GPU 拼接，实现了容量限制的彻底解放。

Stage 3: 记忆交织机制 (Memory Interleave)

针对复杂的多跳推理（Multi-hop Reasoning），单词静态检索往往不足。MSA 自适应地在“生成检索 ID”与“上下文扩展”之间交替迭代。检索到的文档动态转化为下一轮的查询线索，直到证据链完整闭环，模型才启动自回归生成最终答案。

Extrapolation Robustness

< 9%

从 16K 到 1 亿 Token，MSA 展现出了近乎恐怖的抗衰减能力。在 MS MARCO 基准中，其性能下降不到 9%。

94.84%

NIAH Accuracy @ 1M Tokens

在 RULER 大海捞针（Needle-In-A-Haystack）评测中，常规长文本模型（如 Qwen2.5-14B-1M）在跨越 128K 后迅速崩溃，而 MSA 在 1M Token 时仍维持了 94.84% 的惊人精度，甚至优于专门的外部存储智能体（MemAgent-14B）。

𝒪(L)

Linear Computational Complexity

完美解决了算力膨胀问题。无论是训练还是在线推理，MSA 的复杂度均与记忆库容量 L 呈严密的线性关系，彻底规避了稠密注意力 𝒪(L²) 的维数灾难。

SOTA

Dominance in QA Benchmarks

在包含 TriviaQA, DuReader, 2WikiMultiHopQA 等九大长文本 QA 测试中，MSA 以仅仅 4B 的骨干网络参数量，全面超越了使用相同骨干的常规 RAG，并能与包含 235B 甚至 70B 顶尖生成器组合的最佳 RAG 架构（Best-of-breed RAG）相抗衡。

结论与未来

技术终将趋于无形。MSA 并非仅仅是一个对长上下文模型渐进式改良的参数游戏，而是通往通用人工智能（AGI）生命级记忆伴侣的一块核心架构基石。

通过引入具备文档级 RoPE 的稀疏注意力架构、KV缓存压缩以及支持单节点大规模推理的并行设计，MSA 成功实现了**记忆容量与逻辑推理能力的彻底解耦**。正如烧蚀实验（Ablation Study）所严苛证明的：渐进式扩展上下文的课程学习（提升 7.6%）、记忆交织推理（缺失导致复杂任务掉点 19.2%）、生成式检索的预训练，以及保持原生文本语义的完整注入，四者缺一不可，共同铸就了其对抗噪声与遗忘的坚固城墙。

尽管在处理跨越极度分散、隐性强耦合文档的结构化关联时仍面临微小挑战，但这仅仅是攀登更高智能维度的序章。MSA 证明了，在应对无限的信息熵增时，遵循克制、留白与秩序的系统设计（稀疏性），往往比野蛮的算力堆砌（稠密计算），拥有更宏大深远的生命力。
